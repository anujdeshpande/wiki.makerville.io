---
title: Makerville Badge
---

A minimal badge made with an ESP32C3 and a small OLED screen. 

PCB designed using KiCAD by [Anool](https://x.com/anool), <br/>
firmware written using Zephyr by [Anuj](https://anujdeshpande.com), <br/> 
and manufactured using [PCBKingdom.com](https://pcbkingdom.com) by [Akshay](https://www.linkedin.com/in/akshay-sharma-24a1057b/?originalSubdomain=in)

[Source code and design files](https://github.com/Makerville/makerville-badge)
